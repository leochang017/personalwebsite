Project Overview

NapkinNotes is a full-stack EdTech platform built with Python Flask that transforms handwritten and digital notes into AI-powered study resources. The application integrates advanced OCR technology, natural language processing, and a comprehensive marketplace system to create a complete educational ecosystem for students.

Core Platform Features

  • Advanced OCR and PDF text extraction with Google Vision API
  • AI-powered note summarization using Anthropic Claude API
  • Multi-format file upload system (images, PDFs, text documents)
  • Course-specific organization with Princeton Day School integration
  • Real-time note sharing and collaboration system
  • Integrated marketplace for textbooks and school supplies
  • Automated email notification system with SMTP integration
  • User authentication and profile management

Development Journey

Ideation (Aug 2025)

Identified the need for an AI-powered note-taking platform that could help students transform messy handwritten notes into organized, searchable digital resources.

Development Phase (Aug - Sep 2025)

Built full-stack application with Python Flask, integrated Google Vision OCR for text extraction, and implemented Claude API for intelligent summarization. Developed comprehensive marketplace system with email notifications.

Launch & Growth (Sep 2025 - Present)

Successfully launched at Princeton Day School with initial user base. Continuing to iterate on features based on user feedback and expanding platform capabilities.

Impact & Future Development

The platform has successfully demonstrated the value of combining OCR technology with modern AI to enhance the student learning experience.
